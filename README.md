# Deploy a simple contract using Hardhat
## Steps
### 1. Clone Repository
```bash
git clone https://github.com/ToanBm/hardhat-deploy-contract.git && cd hardhat-deploy-contract
```
### 2. Run the setup script
```Bash
chmod +x contract.sh && ./contract.sh
```
